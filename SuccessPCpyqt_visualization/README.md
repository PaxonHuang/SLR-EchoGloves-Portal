# 手势识别可视化系统使用说明

## 系统概述

这是一个基于PyQt5的实时手势识别可视化系统，专为ESP32数据手套项目设计。系统提供了完整的数据采集、可视化、识别和管理功能。

## 功能特性

### 1. 实时数据可视化
- **弯曲传感器监控**: 5个弯曲传感器的实时数据图表
- **IMU传感器数据**: 陀螺仪和加速度计的三轴数据可视化
- **3D可视化**: 基于四元数的手部姿态3D显示（需要OpenGL支持）
- **数据统计**: 实时传感器数据统计信息

### 2. 智能手势识别
- **多种手势支持**: 支持14种静态手势和3种动态手势
- **实时识别**: 毫秒级手势识别响应
- **置信度显示**: 显示识别结果的可信程度
- **稳定性检测**: 过滤不稳定的识别结果
- **历史记录**: 保存识别历史和统计数据

### 3. 数据采集与管理
- **实时录制**: 带标签的数据录制功能
- **多格式导出**: 支持CSV、JSON、NPY、Excel格式
- **自动标签**: 智能标签管理
- **数据预览**: 实时数据预览和统计

### 4. 灵活的系统设置
- **串口配置**: 自动检测和配置串口参数
- **显示设置**: 可调节的更新频率和显示参数
- **识别参数**: 可调节的置信度阈值和稳定性窗口
- **主题支持**: 多种界面主题选择

## 系统要求

### 硬件要求
- ESP32开发板

## 安装步骤

1. **克隆项目**
   ```bash
   git clone <项目地址>
   cd Hand-Gesture-Recognition-main/pyqt_visualization
   ```

2. **安装依赖**
   ```bash
   # 如果使用 Python 3.11
   /usr/local/bin/python3.11 -m pip install -r requirements.txt
   
   # 或者使用默认的 python3
   python3 -m pip install -r requirements.txt
   ```

3. **运行系统测试**
   ```bash
   # 使用 Python 3.11
   /usr/local/bin/python3.11 test_system.py
   
   # 或者使用默认的 python3
   python3 test_system.py
   ```

4. **启动应用程序**
   ```bash
   # 方法1：使用启动脚本（推荐）
   ./launch.sh
   
   # 方法2：直接使用 Python 3.11
   /usr/local/bin/python3.11 start.py
   
   # 方法3：使用默认 python3
   python3 start.py
   ```

## 使用指南

### 1. 首次启动

1. 启动应用程序后，首先进入"系统设置"选项卡
2. 在"串口设置"中选择正确的串口（通常是/dev/ttyUSB0或COM端口）
3. 设置波特率为115200
4. 点击"保存设置"

### 2. 连接设备

1. 确保ESP32设备已正确连接到计算机
2. 点击工具栏中的"连接设备"按钮
3. 状态栏会显示连接状态

### 3. 开始可视化

1. 设备连接成功后，点击"开始可视化"按钮
2. 查看实时传感器数据：
   - **弯曲传感器选项卡**: 显示5个手指的弯曲数据
   - **IMU传感器选项卡**: 显示陀螺仪和加速度计数据
   - **数据信息选项卡**: 显示四元数和统计信息

### 4. 手势识别

1. 右侧面板会显示实时识别的手势
2. 支持的静态手势包括：
   - Cash, Come Here, Excellent, Fingers crossed, Fist
   - Five, Four, Go Away, One, Stop, Three
   - Thumbs Down, Thumbs Up, Two

3. 置信度条显示识别的可信程度
4. 历史记录显示最近的识别结果

### 5. 数据录制

1. 切换到"数据管理"选项卡
2. 在"数据录制"子选项卡中：
   - 选择或输入手势标签
   - 设置录制时长
   - 点击"开始录制"
3. 录制完成后可以保存数据

### 6. 数据导出

1. 在"数据导出"子选项卡中：
   - 选择文件格式（CSV、JSON等）
   - 设置保存路径和文件名
   - 点击"导出当前数据"

## 故障排除

### 常见问题

1. **无法连接设备**
   - 检查USB线缆连接
   - 确认串口设置正确
   - 检查设备驱动程序

2. **OpenGL警告**
   - 3D可视化需要OpenGL支持
   - 如果不可用，系统会自动禁用3D功能
   - 基本功能不受影响

3. **识别准确率低**
   - 调整置信度阈值设置
   - 增加稳定性窗口大小
   - 重新训练模型

4. **数据显示异常**
   - 检查传感器连接
   - 确认数据格式正确
   - 重启应用程序

### 性能优化

1. **降低更新频率**: 在系统设置中调低FPS可以减少CPU使用率
2. **减少数据点**: 降低最大数据点数量可以节省内存
3. **禁用3D可视化**: 如果不需要可以提高性能

## 开发说明

### 项目结构
```
pyqt_visualization/
├── main.py                 # 主程序入口
├── start.py               # 启动脚本
├── test_system.py         # 系统测试脚本
├── requirements.txt       # 依赖列表
├── core/                  # 核心功能模块
│   ├── serial_interface.py   # 串口通信
│   └── gesture_classifier.py # 手势分类器
├── ui/                    # 用户界面
│   ├── main_window.py     # 主窗口
│   └── widgets/           # UI组件
│       ├── sensor_plots.py   # 传感器可视化
│       ├── gesture_display.py# 手势显示
│       ├── data_manager.py   # 数据管理
│       └── settings_panel.py # 设置面板
└── models/                # 机器学习模型
    └── train_model.py     # 模型训练脚本
```

### 扩展开发

1. **添加新手势**: 在gesture_classifier.py中扩展手势列表
2. **新增传感器**: 修改serial_interface.py中的数据格式
3. **自定义UI**: 创建新的widget组件
4. **改进算法**: 在models/目录下添加新的训练脚本
